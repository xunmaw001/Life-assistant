# Life-assistant
基于微信小程序的生活小助手，无需下载安装，打开微信就能用。在生活缴费模块，用户一键完成水、电、燃气缴费，还能设置缴费提醒，避免逾期。提供天气查询服务，精准推送未来多日的天气信息，帮助合理安排出行。日常出行时，借助内置地图，规划公交、驾车等路线。想学习新菜谱，搜索功能提供海量美食制作教程。此外，它还支持日程管理，设置重要事项提醒。这款小程序将诸多生活服务集成一体，让日常生活更轻松便捷 。
